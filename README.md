# homebrew-zyrax

Homebrew tap for [Zyrax Guard](https://github.com/tiagosilva07/zyrax-guard) — audit your AI
agent configs (prompt injection, malicious MCP servers, credential-exfil) before you run them,
and vet the packages they pull in.

## Install

```bash
brew install tiagosilva07/zyrax/zyrax-guard
```

or:

```bash
brew tap tiagosilva07/zyrax
brew install zyrax-guard
```

The formula installs the **signed release binary** (SHA-256 verified by Homebrew) for
macOS/Linux on amd64/arm64. `brew upgrade` picks up new releases.

> `Formula/zyrax-guard.rb` is generated by `scripts/gen-homebrew-formula.mjs` in the main
> repo and auto-bumped on each release — do not edit it by hand.
